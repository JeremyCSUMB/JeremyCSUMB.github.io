How to Buy from a Vending Machine

By Jeremy Shaw


Step 1: Select Visually the Item of Choice

Scan through the different snacks and check whether there is something that might satisfy your cravings. The types of products that you can buy vary from machine to machine; however, most of them include some kind of chips, chocolates, cookies, and gums. Some machines offer healthy snacks such as baked chips, granola bars, and peanuts.

Even though most vending machines contain fresh products, it might be convenient to try to visually double check the expiration date of the snack you want to buy.

Step 2: Get the Item of Choice Identification Code

Once you have selected visually the item you want to buy, get its identification code. This code is located right below the item.

In some vending machines this code is composed of a letter and a number like "C5"; however in some other machines, it is just a number.

If for any reason the identification code is not visible (e.g., the snack itself is covering part of the code), try looking at the codes next to it (to the left, right, above, and below). In most vending machines the codes follow a sequence and by looking at the codes next to one item, it is possible to deduce its code.

Remember this identification code since you'll be using it in the next steps.
